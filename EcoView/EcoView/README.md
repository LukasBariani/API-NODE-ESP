<h2>Tecnologia e Arquitetura</h2>
<p>O Front-End da plataforma EcoView é o componente responsável por transformar os dados brutos da API em informação visual e acionável para o usuário[cite: 331, 333].</p>
<ul>
  <li><strong>Tecnologia Principal:</strong> O sistema foi desenvolvido em **React.js**, uma das bibliotecas mais modernas e flexíveis para criação de interfaces web[cite: 334].</li>
    <li><strong>Performance:</strong> O uso do React.js garante **velocidade, responsividade** e uma experiência fluida em diferentes dispositivos[cite: 334].</li>
   <li><strong>Arquitetura de Comunicação:</strong> A interface web acessa as informações do banco de dados (MySQL) através da **API EcoView (Node.js)**[cite: 289, 350], garantindo que os dados exibidos estejam sempre atualizados.</li>
</ul>

<hr>

<h2>Design e Experiência do Usuário (UX)</h2>
[cite_start]<p>A interface foi pensada para ser intuitiva, agradável e funcional, priorizando a leitura dos dados e a navegação simples[cite: 277, 335].</p>
<ul>
   <li><strong>Identidade Visual:</strong> O design é **minimalista**, seguindo a identidade visual do EcoView, com cores suaves, ícones inspirados na natureza e um layout limpo que valoriza os indicadores ambientais[cite: 335, 336].</li>
   <li><strong>Responsividade:</strong> O Front-End é totalmente **responsivo**, adaptando-se a diferentes tamanhos de tela (computadores, tablets e smartphones), permitindo o acesso aos dados de qualquer lugar[cite: 343].</li>
</ul>

<hr>

<h2>Procedimentos de Análise e Visualização de Dados</h2>
<p>O dashboard é o núcleo da análise, onde os dados coletados pelos sensores da Dispara-Ervilha são processados e apresentados para apoiar a tomada de decisões conscientes[cite: 233, 357].</p>
<ul>
    <li><strong>Visualização em Tempo Real:</strong> Os gráficos, métricas e painéis interativos são **atualizados em tempo real** [cite: 281, 289, 337], refletindo com precisão o que está acontecendo no ambiente monitorado[cite: 351].</li>
    <li><strong>Indicadores Apresentados:</strong> Permite acompanhar as variações de **temperatura, umidade, luminosidade e qualidade do ar** de forma visual[cite: 337].</li>
   <li><strong>Geração de Relatórios e Comparação:</strong> O sistema possibilita a **comparação de informações entre diferentes setores** e a **geração de relatórios automáticos**, facilitando a análise de tendências e a definição de metas de sustentabilidade[cite: 233, 246, 387].</li>
    <li><strong>Gráficos Dinâmicos:</strong> Utilizam bibliotecas de gráficos dinâmicos para tornar a navegação mais envolvente e facilitar o entendimento dos dados, mesmo para usuários não-técnicos[cite: 338].</li>
</ul>
